# Data gathering: Gathering ground truth
To build a ground truth of residential proxy behavior, this project uses a set of Jupyter notebooks.

### Ground-truth JSON datasets
These files are generated by sending controlled requests through residential proxy providers and recording the observed exit IPs and associated metadata. These are not publicly available.

- **prod-brighdata.json**  
  - Provider: Bright Data  
  - Target: `sne25.nl`  
  - Generated by: `gather_proxy_ips.ipynb`  
  - Description: Measurement results for Bright Data proxies connecting to `sne25.nl`.

- **prod--withifconfig-brighdata.json**  
  - Provider: Bright Data  
  - Targets: `ifconfig.me`  
  - Generated by: `gather_proxy_ips.ipynb`  
  - Description: Measurement results with additional IP/metadata obtained via `ifconfig.me`.

- **prod-decodo.json**  
  - Provider: Decodo  
  - Target: `sne25.nl`  
  - Generated by: `gather_proxy_ips.ipynb`  
  - Description: Measurement results for Decodo proxies connecting to `sne25.nl`.

- **prod-withifconfig-decodo.json**  
  - Provider: Decodo  
  - Targets: `ifconfig.me`  
  - Generated by: `gather_proxy_ips.ipynb`  
  - Description: Measurement results with additional IP/metadata obtained via `ifconfig.me`.

### Jupyter notebooks
The following notebooks were used to gather and analyze ground-truth data and baselines.

- **AnalyzeRPAASdata2017.ipynb**  
  Uses the RPaaS dataset together with WHOIS data to:
  - Inspect discovered IPs
  - Compute total volume
  - Derive the fraction of IPs associated with KPN

- **BaselineAnalysis.ipynb**  
  Uses the baseline DNS dataset to compute:
  - Daily number of unique domains
  - Daily query counts  
  Provides reference behavior for *normal* DNS usage.

- **CombinedDnsAndProxyAnalysis.ipynb**  
  Combines DNS data with the Decodo and Bright Data ground-truth datasets to:
  - Show amount of Abuse per IP
  - Analyze reverse DNS (rDNS)
  - Measure overlap between different data sources

- **GatherProxyIPs.ipynb**  
  Actively gathers proxy exit IPs by:
  - Sending simple `ifconfig.me` requests through Bright Data and Decodo
  - Recording the observed exit IPs
  - Fetching WHOIS, rDNS, and abuse information

- **GeoIP analysis.ipynb**  
  Aggregates all collected residential proxy IPs and:
  - Performs GeoIP lookups
  - Draws a simple map of proxy locations to visualize their geographic distribution.
